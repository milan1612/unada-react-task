# Planet Images

Place your planet images in this directory with the following names:

- `etheron.png` - Purple/blue gaseous planet (ETHERON)
- `lumenara.png` - Orange/red fiery planet (LUMENARA)
- `theronix.png` - Green/blue terrestrial planet (THERONIX)
- `orionis.png` - Blue/white oceanic planet (ORIONIS)
- `lumenida.png` - Orange/red gas giant (LUMENIDA)
- `phaeronix.png` - Orange/brown planet (PHAERONIX)

All images should be square or circular planet images that will be displayed as circular elements in the UI.
